What is Parallel Processing?

Parallel processing (or parallel computing) is computation where many calculations are carried out simultaneously instead of sequentially. There are physical constraints to how fast computers can process sequentially, which is why it is important because parallel processing allows computers to run a program in a shorter amount of time. Parallel processing requires programmers to divide tasks into 'subtasks'. Subtasks must not be dependent on the results of other subtasks to be completed. Once all subtasks are completed, their results are combined to produce the desired output. Additional processors can only speed up tasks to the extent that the task has been subdivided.

Parallelism vs. Concurrency

Parallelism is when tasks literally run at the same time, eg. on a multicore processor. Concurrency is when two or more tasks can start, run, and complete in overlapping time periods. It doesn't necessarily mean they'll ever both be running at the same instant.
